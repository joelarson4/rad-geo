rad-geo

a RadReveal addon for Reveal.js

check it out on github!

What does it do?

rad-geo helps you display maps in your presentations.

Why would you want to?

Maps are visually interesting and can add depth to your talk if anything geographic is involved.

How do you use it?

Just install it and add data-rad-geo-* attributes to your slides.

How can you show a specific location?

By using the data-rad-geo-goto attribute:


 <section data-rad-geo-goto="31.2, 121.5">

Can you show a location pin?

Just use the data-rad-geo-pin attribute:








 <section data-rad-geo-pin="31.2, 121.5, Shanghai\, China" 
   data-rad-geo-goto="pin">

Can you show multiple locations?









Use data-rad-geo-pin and seperate locations with a semicolon.

What if you want to zoom in or out?

You can set the data-rad-geo-zoom attribute.


 <section data-rad-geo-zoom="6.8">

What if you want to move faster or slower?

You can control transition speed by setting the data-rad-geo-speed attribute to a number of milliseconds.


 <section data-rad-geo-zoom="6" data-rad-geo-speed="300">

What if you want to move faster or slower?

You can control transition speed by setting the data-rad-geo-speed attribute to a number of milliseconds.


 <section data-rad-geo-zoom="6" data-rad-geo-speed="3000">

Can you use different map styles?

You can select different styles using the
data-rad-geo-style attribute.


 <section data-rad-geo-style="SCENE3D_NASA_BLACKMARBLE">

Can you use a 2D map instead of a 3D globe?

With the Cesium map provider (more on that later) you can use 3D, 2D or "Columbus" modes.


 <section data-rad-geo-style="SCENE2D_NASA_BLACKMARBLE">

What map styles are available?

With the Cesium map provider there are quite a few. Find out more in the documentation.

What is the "Cesium map provider"?

Cesium is the default map rendering library for rad-geo, and it is integrated into rad-geo as the "Cesium map provider".

In the future more map providers might be integrated.

What if I want a slide without a map?

Any slides without a data-rad-geo* attribute will appear without a map.

What if I want a slide without a map?

Any slides without a data-rad-geo* attribute will appear without a map.

Then just adding that attribute on the next slide brings the map back.

Can I display kml?

You can select display kml files with data-rad-geo-kml.


 <section data-rad-geo-kml="kml/Mesopotamia.kml">

How about a kmz file?

You can also display kmz files (compressed kml files) using data-rad-geo-kml.


 <section data-rad-geo-kml="kml/bermuda_triangle.kmz">
